Implemented Features

üéØ 1. Accurate Collision Shapes

Different objects use appropriate collision shapes:

  • Box: Tables, chairs, crates
  • Sphere: Barrels, plants, spheres
  • Capsule: Actors, lamps, cylinders

üîÑ 2. Smooth Sliding Collision Response

Objects slide along walls instead of getting stuck:

if (!blocked) move(targetX, targetZ);

// Try sliding along Z
else if (!blocked) move(currentX, targetZ);

üêõ 4. Debug Visualization

Visual display of collision boundaries:

  • Green wireframe overlays
  • Real-time position updates
  • Toggle via UI button

üîß 5. Collision Layers

Bitmask-based filtering system:

ACTORS: 0b0001 (1)
PROPS: 0b0010 (2)
SCENERY: 0b0100 (4)
STAGE: 0b1000 (8)

Objects only collide if layer bits overlap.
