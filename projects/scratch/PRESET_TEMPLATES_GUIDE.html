The 3D Theater Stage now includes six preset templates for common theatrical scenarios. These templates provide ready-to-use stage configurations with actors, props, lighting, and stage elements.

Technical Details

All preset templates are stored as JSON files in the presets/ directory. They use the Scene Serializer format version 1.0 and include:

  • Scene metadata (name, description, timestamp)
  • Actor positions and rotations
  • Prop types and placements
  • Lighting configurations
  • Camera positions
  • Stage element states (platforms, curtains, scenery panels, etc.)

Creating Custom Presets

You can create your own preset templates:

  1. Set up your stage with the desired configuration
  2. Click "Save Scene" in the UI
  3. Rename your file using kebab-case (e.g., my-custom-scene.json) and move it to the presets/ directory
  4. Edit js/stage.js around line 1270-1277 in the presetSelect.innerHTML section
  5. Add a new option line: <option value="my-custom-scene">My Custom Scene</option>
  6. The value attribute must match your filename (without .json extension)

File Naming Convention: Use lowercase kebab-case for preset filenames (e.g., outdoor-park.json, living-room.json). The dropdown value must exactly match the filename without the .json extension.
